# OwnStep

Безопасный доступ к интернету для Windows на базе Xray-core.

## Быстрый старт

### 1. Настройка
Откройте `setup.html` в браузере и следуйте инструкциям мастера.

### 2. Запуск
```
OwnStep.bat
```

## Возможности

- **VLESS + Reality** — безопасный доступ к интернету
- **Мастер настройки** — простая пошаговая настройка через `setup.html`
- **Импорт ссылок** — вставьте VLESS-ссылку и всё настроится автоматически
- **Ссылки для телефона** — генерация ссылок и QR-кодов для мобильных приложений

## Структура

```
ownstep/
├── setup.html          # Мастер настройки (откройте первым!)
├── OwnStep.bat         # Запуск
├── config.json         # Конфигурация (генерируется мастером)
├── start-vpn.ps1       # Скрипт запуска Xray
├── download-xray.ps1   # Скачивание Xray-core
├── index.html          # Продвинутый интерфейс
└── bin/
    └── xray.exe        # Xray-core (скачивается автоматически)
```
