Покупка сервера

⏱ ~5 минут

Вам нужен VPS (виртуальный сервер) в другой стране. Мы рекомендуем IT-Garage — там можно оплатить российской картой.

1️⃣
Зайдите на it-garage.pro Нажмите «Зарегистрироваться» и создайте аккаунт. Понадобится email.
2️⃣
Выберите «VPS/VDS» в меню Нажмите «Заказать» для VPS-сервера.

Подготовка (Как зайти на сервер)

⏱ ~2 минуты

Прежде чем что-то устанавливать, нужно подключиться к вашему арендованному серверу:

1️⃣
Откройте папку ownstep на компьютере Откройте папку ownstep на вашем компьютере.
2️⃣
Запустите PowerShell В проводнике нажмите «Файл» -> «Запустить PowerShell».
💡
Как работать в PowerShell:
• Чтобы скопировать текст: выделите его мышкой и нажмите правую кнопку мыши (или Enter).
• Чтобы вставить текст: нажмите правую кнопку мыши в любом месте окна.
3️⃣
Введите команду для входа Введите команду: ssh root@IP_ВАШЕГО_СЕРВЕРА (цифры вам даст хостинг). Пример: ssh root@123.45.678.999
4️⃣
Вставьте пароль Вставьте пароль (нажатием правой кнопки мыши) и нажмите Enter (пароль также находится на хостинге).
Важно: При вводе пароля символы не отображаются! Это нормально. Введите вслепую и нажмите Enter. Вас должно пустить на сервер и в начале строки появится root@ЛогинСервера:
1

Оживляем сервер (Установка)

⏱ ~1 минута

Выберите любой удобный для вас способ.

📋 Скопируйте эту команду целиком:
bash <(curl -sL https://raw.githubusercontent.com/tatanika/ownstep/main/server-setup.sh)

Вставьте скопированную команду, нажмите Enter и подождите ~1 минуту. Скрипт сам всё установит и покажет прогресс.

✅
Установка завершена! В самом низу консоли зеленого цвета будет ваша личная ссылка VLESS. Никому её не передавайте (кроме тех, кому хотите дать доступ).
4

Скопировать вашу ссылку

⏱ ~1 минута

После завершения установки скрипт покажет вашу VLESS-ссылку. Она будет выглядеть примерно так:

Пример вашей ссылки (у вас будет своя!):
vless://abc123-def456@12.34.56.78:443?type=tcp&security=reality&sni=www.google.com&pbk=ваш_ключ&flow=xtls-rprx-vision&fp=chrome#OwnStep
📋
Скопируйте эту ссылку! Выделите её мышкой в PowerShell и скопируйте при помощи правой кнопки мыши (или Enter). Или перепишите на бумажку — она понадобится на следующем шаге.
🎉
Серверная часть готова! Сервер будет работать автоматически и переживёт перезагрузку. Больше ничего на сервере делать не нужно.
5

Настройка на компьютере

⏱ ~2 минуты

Теперь настроим VPN на вашем компьютере с помощью мастера OwnStep.

1️⃣
Откройте файл setup.html Он находится в папке OwnStep. Просто дважды кликните по нему — откроется в браузере.
2️⃣
Нажмите «Начать настройку» На первом экране нажмите фиолетовую кнопку.
3️⃣
Вставьте вашу VLESS-ссылку На втором экране выберите «Вставить ссылку» и вставьте ту ссылку, которую скопировали с сервера.
4️⃣
Нажмите «Сгенерировать конфиг» Скачается файл config.json — положите его в папку OwnStep (с заменой, если спросит).
5️⃣
Запустите OwnStep.bat Дважды кликните по файлу OwnStep.bat — VPN включится в фоновом режиме!

🧩 Установка расширения в браузер

Сам по себе VPN уже работает, но чтобы трафик из браузера направлялся через него, нужно установить наше расширение (оно позволит открывать заблокированные сайты через VPN, а российские — напрямую).

🌐
Для Chrome: Перейдите по адресу chrome://extensions/. В правом верхнем углу включите ползунок «Режим разработчика». Нажмите кнопку «Загрузить распакованное расширение» и выберите папку extension-chrome из папки OwnStep.
🔴
Для Яндекс.Браузера: Перейдите по адресу browser://extensions/. В правом верхнем углу включите «Режим разработчика». Нажмите «Загрузить распакованное расширение» и выберите папку extension (находится внутри папки проекта).
🦊
Для Firefox: Перейдите по адресу about:debugging, выберите слева «This Firefox» (или «Этот Firefox») → Нажмите «Load Temporary Add-on» (Загрузить временное дополнение). Выберите файл manifest.json из папки extension-firefox.
✅
Готово! Теперь ваш браузер использует умный VPN. Заблокированные сайты (Instagram, Netflix, Discord и др.) будут открываться автоматически. Вы можете кликнуть на иконку OwnStep в панели расширений браузера, чтобы добавить свои сайты.
📱

Бонус: VPN на телефоне

⏱ ~2 минуты

Ту же ссылку можно использовать и на телефоне!

🤖
Android Установите приложение Hiddify или v2rayNG из Google Play. Откройте приложение, нажмите «+» (в Hiddify — «Новый профиль») → «Добавить из буфера обмена» и вставьте вашу VLESS-ссылку.
🍎
iPhone / iPad Установите приложение Streisand из App Store. Нажмите «+» и вставьте ссылку.
💡
На третьем экране setup.html есть QR-код — отсканируйте его камерой телефона для быстрого импорта настроек!
